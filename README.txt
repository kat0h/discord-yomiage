                    読み上げくん
====================================================

使い方
・!yomiageで起動
・!useroで終了
・起動したチャンネルのpostを読み上げます

python3環境で動作します
必須モジュール
・discord.py
・python-dotenv

インストール
```
pip install -r requirements.txt
```

以下の外部コマンド・パッケージに依存しています
・open-jtalk
・hts-voice-nitech-jp-atr503-m001
・open-jtalk-mecab-naist-jdic

DiscordのAPIキーを.envファイルに書き込んで下さい
